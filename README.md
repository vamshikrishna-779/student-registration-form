# Student Registration Form

## 📌 Project Overview
This is a Student Registration Form built using HTML. It allows users to enter student details through a simple and user-friendly form.

## ✨ Features
- Student name input
- Email input
- Phone number input
- Gender selection
- Course selection
- Address field
- Submit and Reset buttons

## 🛠️ Technologies Used
- HTML

## 📂 Project Structure
student-registration-form/
│── index.html
│── README.md

## 🚀 How to Run
1. Download or clone the repository.
2. Open the project folder.
3. Open `index.html` in any web browser.
